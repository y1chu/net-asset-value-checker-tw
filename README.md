# 基金即時估值 (Taiwan Fund Live NAV Estimator)

每天想買基金前，不用再手動把基金的股票組成一檔檔輸入看盤 App。
輸入基金代碼，它會自動抓取該基金**最新揭露的前十大持股與權重**，
拉**台股即時報價**，算出「今日估算漲跌」，幫你判斷當天要不要進場。
持股每月自動更新，不會再過期。

## 它做什麼

- **搜尋基金**：直接打名稱或代碼（例：「安聯台灣科技」或 `ACDD04`），自動帶出基金
- **常用清單**：點 ★ 把常看的基金加入我的最愛，之後一鍵切換
- 從 MoneyDJ（嘉實資訊）抓基金每月揭露的持股明細（個股名稱＋權重）
- 用證交所 ISIN 名單把中文股名對到股票代碼（上市 / 上櫃）
- 從證交所 MIS 即時報價算每檔漲跌，依權重加權 → **今日估算漲跌 %**
- 手機開網頁即可看，每 30 秒自動更新（切到背景會暫停）

## 限制（重要）

基金公司每月只揭露**前十大持股**（約占淨值 50–70%），不是完整投資組合。
所以這是一個**方向性估算**，不等於基金公司公告的正式淨值 —
但前幾大持股通常主導當日漲跌，足以判斷「今天大概是紅還是綠」。
畫面上會顯示「估算涵蓋淨值 %」讓你知道涵蓋多少。

## 執行

```bash
npm install
npm start
```

- 電腦：開 http://localhost:3000
- 手機（同一個 Wi-Fi）：開 `http://<你電腦的區網IP>:3000`（例如 http://192.168.0.139:3000）

在搜尋框打基金名稱或代碼即可，例如「安聯台灣科技」、「統一黑馬」、`ACDD04`。
選到基金後點 ★ 即可加入常用清單（存在瀏覽器本機）。

> 全站約 4,400 檔國內基金的名稱↔代碼索引，啟動時自動從 MoneyDJ 建立並快取到
> `data/fund-index.json`（每週更新一次），所以搜尋是即時的。

## 結構

```
server/
  index.js      Express；/api/fund/:code（持股＋報價＋估算）、/api/search（基金搜尋）
  holdings.js   抓 & 解析 MoneyDJ 投資明細（Big5 解碼）
  stocks.js     證交所 ISIN 名單 → 中文股名對代碼（快取 12h）
  prices.js     證交所 MIS 即時報價
  fundindex.js  爬 MoneyDJ 建立全站基金名稱↔代碼索引（快取到 data/，每週更新）
public/         手機版單頁前端（index.html / app.js / styles.css）
data/           fund-index.json（自動產生的基金索引快取）
```

## 資料來源

- 持股：MoneyDJ 理財網（嘉實資訊）`www.moneydj.com/funddj/yp/yp013000.djhtm`
- 股名對代碼：證交所 ISIN `isin.twse.com.tw`
- 即時報價：證交所 MIS `mis.twse.com.tw`

僅供個人參考，非投資建議；報價與淨值以官方公告為準。
